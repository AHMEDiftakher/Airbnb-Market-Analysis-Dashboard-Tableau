# Airbnb-Market-Analysis-Dashboard-Tableau
### 1.	Project Title / Headline
A dynamic and interactive Airbnb Market Analysis Dashboard built with Tableau to explore rental listings in Seattle, Washington (2016). The dashboard transforms raw Airbnb data into actionable insights, analyzing property prices, bedroom distribution, zip code trends, and revenue patterns to support academic research and data-driven decision-making.

### 2.	Short Description / Purpose
The Airbnb Market Analysis Dashboard is a visually rich Tableau report that enables users to monitor rental prices, explore neighborhood performance, and evaluate trends across time and property types. It is designed for students, researchers, and analysts seeking insights into short-term rental markets and regional housing trends.

### 3.	Tech Stack
The dashboard was built using the following tools and technologies:<br>

• Tableau Desktop – Primary platform for creating interactive dashboards<br>
• Calculated Fields – Metrics, KPIs, and derived insights<br>
• Filters & Dashboard Actions – Enable interactive exploration and drill-down<br>
• Data Modeling – Relationships and structuring for accurate analysis<br>
• File Format – .twbx for Tableau workbook development and .png for dashboard preview<br>

### 4.	Data Source
The dataset contains Airbnb rental listings in Seattle, Washington (2016):<br>
• Property details including bedroom count, price, and listing type<br>
• Neighborhood and zip code information for geographic analysis<br>
• Revenue and occupancy data over time<br>
The structured data supports price comparison, inventory analysis, and regional trend evaluation.<br>

Sources:
[Inside Airbnb – Seattle Data](https://insideairbnb.com/get-the-data/)

### 5.	Features / Highlights
🔎 Business Problem<br>
Understanding Airbnb rental markets requires analyzing large, complex datasets. Key questions include:<br>
• Which neighborhoods have the highest average prices?<br>
• How does bedroom count affect rental pricing?<br>
• Which areas generate the most revenue?<br>
…are difficult to answer without visual analytics.

🎯 Goal of the Dashboard<br>
To provide an interactive tool that:<br>
• Analyzes rental prices and revenue trends<br>
• Highlights high-performing neighborhoods and property types<br>
• Supports academic research and market evaluation<br>

📈 Walkthrough of Key Visuals<br>
Key KPIs Panel<br>
• Average Price<br>
• Total Revenue<br>
• Total Listings

Average Price per Bedroom (Bar Chart): Shows price trends by property size.<br>
Zip Code Price Comparison (Map/Chart): Highlights neighborhood price differences.<br>
Revenue Trend Analysis (Line Chart): Tracks revenue over the year.<br>
Bedroom Listings Count (Column Chart): Visualizes inventory distribution.<br>
Interactive Filters (Slicers): Filter by bedroom count, neighborhood, and time for deeper insights.<br>

🚀 Business Impact & Insights<br>
• Price Analysis: Understand how property size affects rental pricing<br>
• Regional Strategy: Identify high-demand neighborhoods<br>
• Revenue Insights: Track seasonal patterns and growth opportunities<br>
• Academic Research: Supports studies in short-term rental economics and urban housing trends<br>
